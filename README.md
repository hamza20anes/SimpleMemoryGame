# Simple Memory Game (Java Console)

## Overview
A simple **Memory Game** implemented in **pure Java (console-based)**.  
The goal is to match all pairs of hidden cards in the fewest number of attempts.

This project demonstrates **Object-Oriented Programming (OOP)** principles such as encapsulation, modularity, and class interaction â€” all without using graphical frameworks or external libraries.

---

## Features
- 4x4 board (customizable)
- Randomly shuffled card pairs
- Console-based input for selecting coordinates
- Matching logic with revealed/hidden states
- Attempts counter
- Clean and modular OOP design (`Card`, `Board`, `Game`, `Main`)

---

## Technologies Used
- **Language:** Java 17+
- **Paradigm:** Object-Oriented Programming (OOP)
- **IDE:** Eclipse (any Java IDE works)
